# NeuraConcept

Using knowledge graphs and LLMs to make quality education affordable for Indian schools by building personalized "mental models" for students.

## Products

- [GradeOwl](https://github.com/Dip707/gradeowl) — AI-powered subjective exam grading
- [AI Service](https://github.com/Dip707/ai-service) — Multi-provider vision and embedding microservice

## Development

```bash
npm install
npm run dev
```

Built with React + Vite + TypeScript + D3.
